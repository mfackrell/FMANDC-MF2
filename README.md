# FMandC Website

This folder contains the production-built files for `www.fmandc.com`, downloaded directly from Netlify.

## Source of Truth
These files represent the live version of the website as of January 15, 2026. They were retrieved from Netlify deploy `687abe504115fb823ea6ffd1`.

## Local Development
To view the site locally, you can use the following command:

```bash
npm run serve
```

## Structure
- `assets/`: Compiled JavaScript and CSS.
- `index.html`: The main entry point.
- `package.json`: Monorepo integration and local serving scripts.
